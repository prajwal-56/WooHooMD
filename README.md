# wooHoo for Obsidian 🎉

A simple Obsidian plugin that adds a moment of visual celebration to your task management. 

Whenever you check off a task (`- [x]`), the plugin temporarily plays a GIF or video on your screen to give you a feeling of **Celebration**!

## Features

- **Visual Celebration:** Plays an animation on top of your Obsidian window when a task is completed.
- **Custom Assets:** You provide the GIFs! Point the plugin to any folder in your vault.
- **Smart Timing:** Automatically parses GIF binary data to ensure the animation plays exactly once before gracefully fading out.
- **Non-Intrusive:** The animation floats cleanly in the center of your screen and removes itself automatically.

## How to Use

1. **Install:** Install the plugin and enable it in your Obsidian settings.
2. **Create a Folder:** Create a folder anywhere in your Obsidian vault and drop your favorite celebratory `.gif` files into it.
3. **Configure:** Go to the wooHoo settings tab and type in the path to your folder (e.g., `Assets/GIFs`).
4. **Customize:** You can also configure the size and the fallback duration of the GIFs.
5. **Check a Box:** Click a checkbox in any note and a GIF will play on your screen!

## Settings

You can customize the experience in the plugin settings:
* **GIF Folder Path:** The location of your custom animations.
* **GIF Size:** Adjust how large the animation appears on your screen (in pixels).
* **Fallback Duration:** Set a default duration (in milliseconds) for files that are not GIFs.

## Installation

### Manual Installation
1. Go to the **Releases** section of this repository.
2. Download `main.js`, `manifest.json`, and `styles.css` from the latest release.
3. Place them in a folder called `woo-hoo` inside your `.obsidian/plugins/` directory.
4. Reload Obsidian and enable the plugin.
